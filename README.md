## Project Overview

This repository contains the complete Digital Literacy Portfolio developed as part of the Student Digital Ambassador role. The project demonstrates proficiency across five core modules of digital literacy, ranging from visual communication and professional networking to coding collaboration and cyber-safety.

## Module Summaries
# Task 1: Digital Literacy Awareness Infographic Objective: Create a visual resource to help peers understand digital literacy and safe internet practices.Tools Used: Canva.
 Summary: A one-page infographic covering essential digital tools for students, safe internet habits, and professional online presence.

# Task 2: Student Digital Portfolio Objective: Establish a professional digital footprint across industry-standard platforms.Platforms: GitHub, LinkedIn, and Kaggle.
 Summary: Developed a foundational professional presence, including a customized GitHub profile README and updated academic credentials on LinkedIn.

# Task 3: Coding and Cloud Collaboration Objective: Demonstrate familiarity with coding practice environments and cloud-based collaborative tools.Platforms: HackerRank and Google Workspace.
 Summary: Completed beginner-level coding challenges and designed a 5-question "Digital Literacy Awareness Quiz" using Google Forms to engage batchmates.

# Task 4: Professional Email & Social Media Etiquette Objective: Master professional communication standards for academic and corporate environments.
 Deliverables: Professional email drafts for assignment extensions and internship inquiries, along with a Social Media "Do’s and Don’ts" checklist for college students.

# Task 5: Cybercrime Awareness & Prevention Objective: Research online threats and produce educational resources for peer safety.
 Summary: Developed a detailed case study on phishing/cybercrime and a comprehensive "Stay Safe Online" prevention guide, including national reporting channels like 1930 and the National Cyber Crime Portal.
